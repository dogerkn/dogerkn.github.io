# Patch notes for 2.10a - Bug squashing

Three BIG bugs squashed! All three have been around for quite some time. Releasing these updates a bit faster, with less testing beforehand - please come complain on discord if you find new issues!

## What's new:


+ **Fixed Earth UI appearing after loading a save elsewhere** — the wrong theme could show up depending on your progress, this should now resolve correctly

+ **Fixed a soft-lock on mobile when looting** — rapidly tapping the loot button during the opening animation could permanently block all clicks, requiring a refresh

+ **Fixed upgrade search stalling** in late gameplay — the search could get stuck and stop finding new upgrades
    - Search Doges now smells all

## What's next:

+ Continuing to hunt down bugs and edge cases before a major engine upgrade (PixiJS 4 to 8) — if you run into anything weird, let us know!
